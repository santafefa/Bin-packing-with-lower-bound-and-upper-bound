# Bin-packing-with-lower-bound-and-upper-bound
Solving NP hard Bin packing with lower bound and upper bound problem with Evolutionary Computation Algorithms and finetune each algorihm with optuna, also visualization the evolution progress of the population after each generation to find out which algorithm is the best for each 
separate goal.
EC algorithms used are:  
 GA  
 GP  
 EP  
 SG-ES  
 CMA-ES  
 ANT COLONY  
 MEMETIC ANT COLONY WITH HILL CLIMBING  
 DE WITH HYBRID POPULATION INITALIZATION AND LOCAL SEARCH
